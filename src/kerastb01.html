Tuberculosis Prediction

Chest x-rays are a very common imaging tool in healthcare. This means the ability to pick up tuberculosis from a chest x-ray is something that may be beneficial to patients. Eventually we may expand this model to identify other conditions from chest x-rays.

In a previous test here we used a convolutional neural network to classify x-rays into positive or negative for TB. It was based on the python library pytorch.

This model is an attempt to improve the first model built on the deep learning library keras here.

Conclusion

The project aims to achieve an accurate and reliable classification model for TB detection in chest X-ray images.

The notebook above shows the use of a keras based convolutional neural network to classify chest x-rays based on the presence or abscence of tuberculosis. We can view it in comparison to the pythorch based model here. This model is an improvement on the first keras model here. There is still a lot left to do to bump those numbers up.

In the next phase, we will explore the use of pretrained models and assess their performance at this task.
